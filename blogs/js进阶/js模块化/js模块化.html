# 模块化规范

# CommonJS

# 规范

# 说明

  • 每个文件都可以当作一个模块
  • 服务器端：模块的加载是运行时同步加载的（阻塞）
  • 在浏览器端：模块需要提前编译打包处理(等待时间长 用户体验差)

# 基本语法

  • 暴露模块

    • module.exports = value
    • exports.xxx = value
    • 问题：暴露的模块到底是什么？---exports 对象（exports 默认为{}）

  • 引入模块

    • ​ require(xxx)

      ​ 第三方模块：xxx 为模块名

      ​ 自定义模块：xxx 为模块文件路径
